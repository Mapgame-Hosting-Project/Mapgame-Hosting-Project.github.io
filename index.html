Mapgame Hosting Project (MHP)

The Mapgame Hosting Project aims to simplify the creation and management of mapgames.

Resources and software

We have many tools for mapgamers to use to enhance their experience.

If you want to host a mapgame with minimal fuss, choose our web option.

If you want to create a mapgame on Discord or integrate with an existing one on the platform, use our Discord bot.
